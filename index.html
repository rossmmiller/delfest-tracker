The rules:
· Three clean reps before raising the tempo. Never just run full-speed sloppy.
· Two bars clean → rest → repeat beats 32 bars sloppy every time.
· Most Delfest jams run 90–110 bpm. Chop at 90 before worrying about breaks at 100.
· Auto-speedup in Strum Machine is perfect for this — set it to +2 bpm per rep.
